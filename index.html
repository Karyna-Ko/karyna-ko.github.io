Мене звати Карина

Я веб-розробник із міста Дніпро. У 2017 року закінчила Дніпровський національний університет імені Олеся Гончара (факультет української та іноземної філології та мистецтвознавства) за спеціальністю Філологія. Романські мови та літератури (французька мова). Працювала копірайтером та перекладачем на фрілансі.
У 2021 році почала навчатися веб-розробці та закохалася у цю сферу.
Створювати сайти – це окремий вид мистецтва, до якого слід підходити скрупульозно та відповідально, але водночас не втрачати нотку творчості!

Навички

Що я використовую у роботі

HTML5

Саме він створює каркас вашого сайту або програми, а п'ята версія дозволить мені створювати більш SEO-оптимізовану структуру вашого продукту

CSS3

Ця мова стилів дозволяє мені створювати абсолютно будь-який зовнішній вигляд вашого сайту або програми. Все обмежується лише вашою фантазією!

JavaScript

Ця мова програмування дозволяє оживити все, що завгодно: слайдери, вікна, підказки, вкладки, отримання даних від сервера та багато іншого

Jquery

Швидка, невелика та багатофункціональна бібліотека JavaScript. Значно спрощує обхід та маніпуляції з HTML-документами, обробка подій, анімація та Ajax, завдяки легкому у використанні API, який працює в багатьох браузерах

React

Ця бібліотека дозволяє створювати веб-додатки. Ми можемо створити максимально інтерактивний продукт саме під ваші цілі

Redux

Redux — відкрита JavaScript бібліотека призначена для керування станом програм. Найчастіше використовується разом з React або Angular для побудови інтерфейсів користувача.

Typescript

TypeScript — мова програмування, представлена Microsoft восени 2012; позиціонується як засіб розробки вебзастосунків, що розширює можливості JavaScript.

Material UI

Material UI — одна з найбільш популярних бібліотек компонентів React. Компоненти та всі шаблонні елементи створені відповідно до принципів Google Material UI, але водночас вони є компонентами React.
